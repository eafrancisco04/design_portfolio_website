Technical Drawing

This project is a showcase of our technical ability in Adobe Illustration. The objective was to recreate a household appliance using gradient mesh. Image trace and Photoshop cannot be used to draw the device. The device chosen is a 3 in 1 breakfast machine. I had tried to make the machine as realistic as possible, which I believe turned into a success.

Part of the assignment also was to put the machine in a simple advertisement. In this case, we can use Photoshop to place it in an ad and add shadows when needed.
